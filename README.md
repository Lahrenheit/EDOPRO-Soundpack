# EDOPRO-Soundpack

Sound pack by Pastisha for EDOPRO

Credits to original creators

Discord: Pastisha#8564

## Setup

in configs.json write

```
{
     			"url": "https://github.com/Pastisha/EDOPRO-SoundPack",
      		        "repo_name": "Pastisha EDOPro Soundpack",
     			"repo_path": "./sound/",
     			"should_update": true,
     			"should_read": true
},
```
