# EDOPRO-Musicpack

Music pack by Pastisha for EDOPRO

Credits to original creators

Discord: Pastisha#8564

## Setup

in configs.json write

```
{
     			"url": "https://github.com/Pastisha/EDOPRO-MusicPack",
      		        "repo_name": "Pastisha EDOPro Musicpack",
     			"repo_path": "./sound/",
     			"should_update": true,
     			"should_read": true
},
```
